# Genshin Map Downloader

Dark-theme GUI downloader for Teyvat map tiles.

## Features
- Auto version detection
- Region selection
- Resume support
- Retry system
- Progress bar
- Live log panel

## Run
pip install -r requirements.txt
python main.py
